# Magento 2 Blog Extension

[Magento 2 Blog](https://www.ulmod.com/magento-2-blog-pro-extension.html)  is a Magento 2 extension that allows you to create any number of blog posts/articles to communicate with your visitors and increase traffic to your site.

With Magento 2 Blog, you can create any number of posts/articles, group them per category or tag, and display them at the storefront.

<a href="https://www.ulmod.com/magento-2-blog-pro-extension.html"><img width="205" src="https://www.ulmod.com/media/wysiwyg/um-download-now1.png"></a>

## Key Features

* Create any number of blog posts, categories, and tags
* Support 5 pages layouts: empty, 1 column, 2 columns with left bar, 2 columns with right bar, and 3 columns
* Support list and grid layouts to display posts in the storefront
* Flexible blog sidebar with rich content (search, categories, recent posts, archive, tag cloud, RSS, Social Sharing tool, and custom blocks)
* Support featured posts block at the top of the blog page
* Support widget to display recent blog posts at any pages and positions
* Support related posts and products on the post view page. 
* Ability to display related blog posts on the product page
* SEO optimized: permalinks, SEO-friendly URLs, open graph (OG), meta title, description, and keyword
* Support multiple comment types: Disqus, Facebook, and custom comments
* Mobile-friendly
* Support multiple-stores
